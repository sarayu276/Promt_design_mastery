# Promt_design_mastery
Business Use Case 1 - Customer Support
Zero-shot Prompt
Plain text
You are a professional customer support executive.

A customer says:
"My order has not arrived even after 7 days."

Write a polite response.
Few-shot Prompt
Plain text
Example 1

Customer:
I received a damaged product.

Response:
We sincerely apologize. Please share your order ID and images of the damaged product so we can replace it immediately.

Example 2

Customer:
My payment failed but money was deducted.

Response:
We are sorry for the inconvenience. Kindly share your transaction ID so we can verify and refund if applicable.

Now respond to:

Customer:
My order has not arrived even after 7 days.

Business Use Case 3 - Technical Debugging
Zero-shot Prompt
Plain text
Find the error in this Python code and explain how to fix it.

for i in range(5)
print(i)
Few-shot Prompt
Plain text
Example

Input:

if x==5
print(x)

Output:
Missing colon after if statement.

Now solve:

for i in range(5)
print(i)
Business Use Case 4 - Email Writing
Zero-shot Prompt
Plain text
Write a professional leave email requesting two days of leave due to fever.
Few-shot Prompt
Plain text
Example

Reason:
Family Function

Email:
Dear Sir,
I request leave for one day due to a family function.

Now write for:

Reason:
Fever
Duration:
2 days
